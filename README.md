# Premium Portfolio - Nanda Tarmidzi Abdillah

Tugas Akhir Dasar Desain Antar Pengguna yang merancang platform portofolio digital interaktif berkonsep modern *Glassmorphism* dengan fokus pada topik *Cyber Security*.

## Tech Stack & Libraries
- **Struktur & Desain:** HTML5, CSS3 (Custom Variables, Media Queries)
- **Interaktivitas:** Vanilla JavaScript
- **Pustaka Eksternal:**
  - [Particles.js](https://github.com/VincentGarreau/particles.js/) - Animasi latar belakang interaktif jaringan siber.
  - [Typed.js](https://github.com/mattboldt/typed.js/) - Efek animasi mengetik dinamis pada hero title.
  - [Boxicons](https://boxicons.com/) - Set ikon komponen web modern.

## Fitur Utama
1. **Custom Mouse Cursor:** Efek kursor lingkaran dinamis yang merespons elemen hover.
2. **Scroll Spy Navigation:** Navigasi menu yang otomatis aktif menyesuaikan posisi scroll layar.
3. **Form Mockup & Validation:** Simulasi pengiriman pesan interaktif dengan visual feedback sukses.
4. **Fully Responsive:** Optimasi tata letak yang adaptif di layar desktop, tablet, hingga smartphone.
